在 VS Code 中開啟並建立一個新的 Python 專案

Python 以其簡單易讀的語法和強大的功能，在程式設計新手和專業開發者中廣受歡迎。這篇文章將帶你一步步了解如何在 Visual Studio Code (VS Code) 中建立一個新的 Python 專案，並撰寫你的第一個 Python 程式。

1. 安裝 Python 和 VS Code

開始之前，確保你已經安裝好 Python 和 VS Code：

  • 安裝 Python：訪問 Python 官方網站 下載並安裝最新版的 Python。
  • 安裝 VS Code：前往 VS Code 官方網站 下載並安裝適用於你系統的版本。

小提示：安裝 Python 時，記得勾選 “Add Python to PATH”，以確保在終端中可以正確執行 Python 指令。

2. 安裝 Python 擴展

要在 VS Code 中順利開發 Python 專案，還需要安裝 Python 擴展：

  • 打開 VS Code，點擊左側的 擴展 (Extensions) 圖標。
  • 搜索 “Python” 並安裝由 Microsoft 提供的 Python 擴展。這個擴展包含語法高亮、智能提示、程式碼調試等功能。

3. 建立一個新的資料夾作為專案目錄

接下來，我們需要為 Python 專案建立一個目錄：

  • 在電腦上建立一個新的資料夾，命名為 MyPythonProject 或其他你喜歡的名稱。
  • 打開 VS Code，選擇 File -> Open Folder，然後選擇你剛建立的資料夾作為專案目錄。

4. 建立 Python 檔案

在專案目錄中創建你的第一個 Python 檔案：

  • 點擊 VS Code 左上角的 New File 按鈕，並將檔案命名為 main.py。
  • 在 main.py 中撰寫以下簡單的程式碼：
print("Hello, World!")

這段程式碼將在終端中輸出 “Hello, World!"，是程式設計中的經典入門範例。

5. 選擇 Python 解譯器

為了執行 Python 程式，我們需要選擇正確的 Python 解譯器：

按下 Ctrl + Shift + P，然後輸入 Python: Select Interpreter。 從列表中選擇你已安裝的 Python 解譯器。如果未顯示解譯器，請確認 Python 安裝無誤，並設置了系統環境變數。

6. 執行 Python 程式

現在，你已經可以執行你的第一個 Python 程式了：

打開 main.py 檔案，按下 Ctrl + F5，或右鍵點擊檔案並選擇 “Run Python File”。VS Code 將在終端中執行程式，並輸出 Hello, World!。

Python 的輸入與輸出：print() 函數

print() 函數是 Python 中最常用的輸出函數，用來將資料輸出到螢幕上。這裡有一些常見的 print() 用法：

1. 輸出字串

print("Hello, World!")

這將在螢幕上顯示字串 “Hello, World!"。

2. 輸出數字

print(123)

這行程式碼會輸出數字 123。

3. 輸出多個值

print() 函數可以同時輸出多個值，並且會在每個值之間自動加上空格：

print("Python", "is", "fun!")

這會輸出 Python is fun!。

4. 使用變數

你可以使用 print() 函數輸出變數的值：

name = "Alice"
print("Hello,", name)

這會輸出 Hello, Alice。

print() 是 Python 中極為靈活且重要的函數，能幫助你輕鬆地將數據輸出到螢幕，適合用於學習和開發過程中的調試和展示結果。

小結

你已經完成了在 VS Code 中建立並執行第一個 Python 專案的步驟，並了解了如何使用 print() 函數進行簡單的輸出操作。這是邁入 Python 世界的第一步，接下來你可以開始探索更多的程式設計技巧與應用範例。

繼續保持學習動力，Python 的應用無窮無盡，無論是網頁開發、資料分析、人工智慧，還是自動化程式設計，都有豐富的資源和機會等著你！
